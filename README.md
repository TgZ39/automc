# automc

A CLI tool for downloading and setting up Minecraft servers with different distributions. These are the currently supported distributions:
- Vanilla
- Fabric
- Folia
- Paper
- Purpur
- Velocity
- Spigot (Experimental)


## Installation

Via command line:

```sh
cargo install automc
```
## Usage

Run `automc` in terminal.
